# M√©todos `.push()` y `.pop()`

El m√©todo `push()` a√±ade uno o m√°s elementos al final de un array y devuelve la nueva longitud del array.

El m√©todo `pop()`, por otro lado, elimina el √∫ltimo elemento de un array y lo devuelve. Ambos m√©todos modifican el array original.

## Sintaxis


### `.push()`

- A√±ade los elementos especificados al final del array.
- Retorna la nueva longitud del array.

### `.pop()`

- Elimina el √∫ltimo elemento del array.
- Retorna el valor del elemento eliminado.

## Ejemplo

```jsx

// A√±adiendo un elemento al final del array

// frutas = ['manzana', 'banana', 'naranja']

// Eliminando el √∫ltimo elemento del array

// ultimaFruta = 'naranja'
// frutas = ['manzana', 'banana']
```

## Usos comunes

- **`.push()`** se utiliza para agregar elementos a una lista din√°mica, como una colecci√≥n de objetos o resultados.
- **`.pop()`** se usa para quitar el √∫ltimo elemento cuando ya no es necesario o para implementar estructuras de datos como pilas (LIFO).
