# Método `.sort()`

El método `sort()` ordena los elementos de un array y devuelve el array. La forma en que se ordenan los elementos depende de la implementación de la función de comparación.

## Sintaxis

```jsx
array.sort((a, b) => {
  // Lógica de comparación
});
```

El método `sort()` recibe una función de comparación que compara dos valores. Esta función debe retornar:

- Un número negativo si el primer valor debe aparecer antes que el segundo.
- Un número positivo si el segundo valor debe aparecer antes que el primero.
- Cero si son equivalentes.

## Ejemplo

```jsx
const numbers = [3, 1, 4, 1, 5, 9];

numbers.sort((a, b) => a - b);

// [1, 1, 3, 4, 5, 9]
```

## Usos comunes

- Ordenar números de menor a mayor o viceversa.
- Ordenar cadenas de texto alfabéticamente.
- Ordenar objetos en función de una propiedad.

Dado el siguiente array de nombres de ciudad, ordenalos por longitud de nombre de menor a mayor.

<Playground
