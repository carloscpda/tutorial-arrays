# M√©todos `.shift()` y `.unshift()`

El m√©todo `unshift()` a√±ade uno o m√°s elementos al principio de un array y devuelve la nueva longitud del array.

El m√©todo `shift()`, por otro lado, elimina el primer elemento de un array y lo devuelve. Ambos m√©todos modifican el array original.

## Sintaxis

```jsx
const longitudDelArrayModificado = array.unshift(elemento1, ..., elementoN);
const primerElemento = array.shift();
```

### `.unshift()`

- A√±ade los elementos especificados al principio del array.
- Retorna la nueva longitud del array.

### `.shift()`

- Elimina el primer elemento del array.
- Retorna el valor del elemento eliminado.

## Ejemplo

```jsx
const frutas = ["manzana", "banana"];

// A√±adiendo un elemento al principio del array
frutas.unshift("pera");
// frutas = ['pera', 'manzana', 'banana']

// Eliminando el primer elemento del array
const primeraFruta = frutas.shift();
// primeraFruta = 'pera'
// frutas = ['manzana', 'banana']
```

## Usos comunes

- **`.unshift()`** se utiliza para agregar elementos al principio de una lista, √∫til para estructuras como colas.
- **`.shift()`** se usa para eliminar el primer elemento cuando ya no es necesario o para implementar estructuras de datos como colas.
