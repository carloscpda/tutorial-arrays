# Propiedad `.slice()`

El método `.slice()` se utiliza para crear una copia superficial de una porción de un array en un nuevo array objeto seleccionado desde el inicio hasta el final (no inclusivo) del array original. Este método no modifica el array original.

## Sintaxis

```jsx
array.slice(inicio, fin);
```

### Parámetros

- `inicio`: El índice donde comienza la extracción. El valor por defecto es 0. 
- `fin`: El índice donde termina la extracción (no inclusivo). El valor por defecto es la longitud del array.

## Ejemplo

```jsx
const frutas = ['manzana', 'banana', 'naranja', 'pera'];

// Extraer un segmento del array desde el índice 1 hasta el 3 (excluyendo 3)
const frutasSeleccionadas = frutas.slice(1, 3);  // ['banana', 'naranja']

// Extraer desde el índice 2 hasta el final del array
const frutasFinales = frutas.slice(2);  // ['naranja', 'pera']

// Extraer una copia completa del array
const todasLasFrutas = frutas.slice();  // ['manzana', 'banana', 'naranja', 'pera']
```

## Usos comunes

`.slice()` es útil para obtener subarrays, trabajar con porciones de arrays sin alterar el array original, y para realizar copias superficiales de arrays.
