# Métodos `.shift()` y `.unshift()`

El método `unshift()` añade uno o más elementos al principio de un array y devuelve la nueva longitud del array. El método `shift()`, por otro lado, elimina el primer elemento de un array y lo devuelve. Ambos métodos modifican el array original **in place**.

## Sintaxis

```jsx
array.unshift(elemento1, ..., elementoN);
array.shift();
```

### `.unshift()`
- Añade los elementos especificados al principio del array.
- Retorna la nueva longitud del array.

### `.shift()`
- Elimina el primer elemento del array.
- Retorna el valor del elemento eliminado.

## Ejemplo

```jsx
const frutas = ['manzana', 'banana'];

// Añadiendo un elemento al principio del array
frutas.unshift('pera');  // ['pera', 'manzana', 'banana']

// Eliminando el primer elemento del array
const primeraFruta = frutas.shift();  // ['manzana', 'banana']
```

## Usos comunes

- **`.unshift()`** se utiliza para agregar elementos al principio de una lista, útil para estructuras como colas.
- **`.shift()`** se usa para eliminar el primer elemento cuando ya no es necesario o para implementar estructuras de datos como colas (FIFO).
