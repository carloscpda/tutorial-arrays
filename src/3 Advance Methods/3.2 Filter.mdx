# Método .filter()

El método `filter()` crea un nuevo array con todos los elementos que cumplan con la condición implementada por la función dada.

## Sintaxis

```jsx
const nuevoArray = array.filter((currentValue, index, originalArray) => {
  // Retorna true o false
});
```

Si la función retorna `true`, el elemento actual se incluirá en el nuevo array. Si retorna `false`, el elemento actual no se incluirá.

## Parámetros

- `currentValue`: El elemento actual que se está procesando en el array.
- `index` (Opcional): El índice del elemento actual que se está procesando en el array.
- `originalArray` (Opcional): El array sobre el que se llamó `filter`.

## Ejemplo

```jsx
const numbers = [1, 2, 3, 4, 5];

numbers.filter((number) => number >= 0);
```

## Usos comunes

- Filtrar elementos de un array que cumplan con una condición.
- Eliminar elementos no deseados de un array.

Dado el siguiente array de numeros, crea un nuevo array con los numeros pares.
