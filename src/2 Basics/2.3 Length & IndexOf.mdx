# Propiedad `.length` y Método `.indexOf()`

La propiedad `length` devuelve el número de elementos en un array. El método `indexOf()` busca un elemento en el array y devuelve su posición (índice). Si el elemento no se encuentra, devuelve `-1`. Ambos son muy útiles para trabajar con arrays.

## Sintaxis

```jsx
array.length;
array.indexOf(elemento);
```

### `.length`
- Retorna el número de elementos presentes en el array.

### `.indexOf()`
- Busca el elemento en el array.
- Retorna el índice del elemento si es encontrado, o `-1` si no está presente.

## Ejemplo

```jsx
const frutas = ['manzana', 'banana', 'naranja'];

// Obtener la longitud del array
const cantidadFrutas = frutas.length;  // 3

// Buscar el índice de 'banana'
const indiceBanana = frutas.indexOf('banana');  // 1

// Intentar buscar un elemento que no está en el array
const indicePera = frutas.indexOf('pera');  // -1
```

## Usos comunes

- **`.length`** se utiliza para determinar cuántos elementos hay en un array, útil para bucles y validaciones.
- **`.indexOf()`** se usa para encontrar la posición de un elemento en un array, por ejemplo, para realizar operaciones condicionales basadas en su presencia.
